Add a "Shooting Mode" feature — a multi-step guided workflow for tracking a live studio shoot day. Add a camera icon button in the nav bar to enter Shooting Mode.
STEP 1 — Session Name Generator:
Show big tappable buttons in sequence. First pick Brand: DOPESNOW or MONTEC. Then Gender: MEN or WOMEN. Then Product Type using these short names: JACKET (Snowboard Jacket), PANTS (Snowboard Pants), BASELAYER (Base Layer Pant + Base Layer Top), FLEECE (Fleece Hoodie + Fleece Sweater), BEANIE (Beanie), FACEMASK (Facemask), GLOVES (Ski Gloves), GOGGLE (Ski Goggle), MITTENS (Snow Mittens), LENS (Replacement Lens Ski). After all three are selected, auto-generate the session name in this format: BRAND_GENDER_TYPE_FW[last 2 digits of year]_DD.MM — for example DOPESNOW_MEN_JACKET_FW26_20.03. Show the generated name prominently with a copy-to-clipboard button.
STEP 2 — Product Selection:
Pre-filter products based on the brand, gender, and product type selected in Step 1. Show a model/shortname filter so they can pick which model to shoot (e.g. Adept, Blizzard). If the product type is a grouped type (BASELAYER or FLEECE), show products from BOTH sub-types (e.g. both Base Layer Pant and Base Layer Top) and prompt "Also include [the other sub-type]?" with a yes/no. Let the user select which products to include in this session. Exclude any products flagged as factory delayed.
STEP 3 — Live Tracking Page:
Show the session name at the top with a copy button. List all selected products with checkboxes. When entering this page, change all selected products' status to "In the Studio" (new status — add this to the pipeline). As each product is ticked off, change its status to "Ready for Selection" (new status — add this to the pipeline). Show a progress counter: "5/12 shot".
STEP 4 — Session Complete:
When all products are ticked, or the user clicks "End Session", show two options: "Continue with different products" or "Close Session". If continuing, go back to Step 2 but keep the same session name. This time ALL product types for the same brand are available (not locked to the original type). If closing, any products that were NOT ticked off revert to their previous status before the session started.
Also update the production pipeline to this order: Not Started → In the Studio → Ready for Selection → Ready for Retouch → In Post Production → Post Production - Done → Ready for Upload → Uploaded. Update all status dropdowns, filters, progress bars, and badge colors throughout the app.
